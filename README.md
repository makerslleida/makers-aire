# Makers Aire

Proyecto de ciencia ciudadana impulsado por la Associació Makers Lleida.

## Objetivo

Crear una red abierta de estaciones de monitorización ambiental de bajo coste basadas en ESP32.

## Hardware

- ESP32 DevKit V1 38 pines
- SPS30
- BME280
- Alimentación 5V

## Coste aproximado

20 €
